# Ring Configurator

An interactive 3D ring configurator with dynamic visualization and customization options.

## Project Structure

```
.
├── index.html                    # Main entry point
├── support.js                    # Core functionality
├── assets/                       # Static assets
├── uploads/                      # Upload directory
├── screenshots/                  # Screenshots
└── vercel.json                   # Vercel configuration
```

## Deployment

This project is deployed on Vercel and is ready for production use.

### Local Development

Simply open `index.html` in your browser or use a local server:

```bash
# Using Python 3
python -m http.server 8000

# Using Node.js http-server
npx http-server
```

Then navigate to `http://localhost:8000`

## Features

- Interactive 3D ring visualization
- Multiple collection tabs
- Quick view options
- Fullscreen mode
- Responsive design
- Wishlist functionality
